Proposal
For the development of this project we generated different ideas, so we decided that the best one could be about making an app that would help kids (7 - 12 years) to keep a healthy life by teaching them how to develop correct eating habits, combining them with the proper amount of time for exercise everyday.

The app will let the user introduce his/her health information (weight, height, diseases, etc.) to display an agenda, with different alarms and reminders, that would inform the user about the correct amount of carbs, proteins, fats, and vitamins to eat everyday, combined with a certain type of exercise. All this information would be displayed in a friendly way for the kids to be read, using funny phrases, certain famous cartoon characters, and also bright colors. In this way, the interaction for the user would be more easy and nice.

The main proposal of this project is the use of basic programming to create this app and make it compatible with the most popular devices nowadays, such as smartphones, IPods, IPads, and computers.
